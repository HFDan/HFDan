# whoami

I don't like it when my name is posted on the Internet, so "Dani" is as much as you're gonna get.

I started taking an interest in computers when I was about 7 or 8. Most games were too demanding for my poor Core 2 Duo E6320 and Radeon X550 XT, so I had to get "creative" with the fun I could have.

I started by modding GTA San Andreas, studying how Counter Strike 1.6 and GoldSrc worked, and writing random batch scripts (as any script kiddy would do), but, to be fair, I had no idea what I was doing at that time...

My interest in computers started to take off when I was in 5th grade (10-11 years old). I started learing C++ with AVR microcontrollers (Arduino UNO), and I instantly fell in ❤️ with programming.

As of writing this, it's 2022 and i'm almost 17. By now I've dabbled in almost every type of programming I can think of. From Python scripts, to Assembly and OS development. Recently I've also taken an interest in Binary Exploitation and Reverse Engineering, but I'm not as good at that as I wish.
